# Quiz Application

## Description

A simple Java console-based quiz application.

## Features

* Multiple Choice Questions
* Score Calculations
* Instant Result
* Console-based Interface

## Technologies Used

* Java
* OOP Concepts
* ArrayList
* Scanner Class

## How to Run

1. Open terminal inside src folder

2. Compile files:

```bash
javac *.java
```

3. Run project:

```bash
java Main
```

## Example Output

### Quiz Question

```text
What is the capital of India?

A. Mumbai
B. Delhi
C. Pune
D. Chennai

Enter Answer: B

Correct Answer!
```
